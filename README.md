
## Getting Started


### Installation

Example instalation

1. Clone the repo
   ```sh
   git clone https://github.com/StecBartlomiej/complex_events.git
   ```
2. Create virutal enviroment
   ```sh
   python3 -m venv .vevn
   ```
3. Source virtual enviroment
   ```sh
    source .venv/bin/activate
   ```
4. Install [pytorch](https://pytorch.org/get-started/locally/)

5. Install rest of packages using pip3
```sh
    pip3 install -r requirements.txt
```
